A Django-based web application that allows users to:

- Select location
- View available movies
- View shows by location
- Select seats
- Confirm booking

## Tech Stack
- Python
- Django
- SQLite
- HTML & CSS

## How to Run

1. Clone the repository
2. Create virtual environment
3. Install Django
4. Run migrations
5. Start server
6. Admin Access: Use `python manage.py createsuperuser` to create an admin account. 
  The admin can log in to `/admin` panel and add/manage locations, movies, and shows.


## Features
- Location-based movie filtering
- Seat selection system
- Booking confirmation page
- Clean UI
